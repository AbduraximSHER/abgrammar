# ABGrammar — Master English Grammar

Interactive English grammar learning platform with 94 lessons across 5 levels.

## Tech Stack
- React 19 + Vite
- Tailwind CSS v4
- React Router v7
- Framer Motion
- Lucide React Icons

## Features
- 📚 94 interactive grammar lessons
- 🗺️ Structured learning roadmap (5 levels)
- 🧠 Multiple exercise types (MCQ, error correction)
- 📊 Progress tracking with XP & streaks
- 🌙 Dark mode support
- 📱 Fully responsive

## Getting Started
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
